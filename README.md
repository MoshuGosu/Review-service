# Review-service
Review service system där användare kan recensera böcker och uppdatera vad dem tycker om böcker som dem läst.
# Review Service API

Web API for managing book reviews.

## Tech stack
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- Swagger

## Endpoints

GET /api/BookReviews  
POST /api/BookReviews  
PUT /api/BookReviews/{id}  
DELETE /api/BookReviews/{id}

## Run locally

dotnet run
